# Flask meme site

Displays a picture from meme api.

### How to run

Make sure you have Python 3 and pip installed.

Create and activate venv from parent directory
```bash
python3 -m venv flask-practice
source bin/activate
```

Install dependencies
```bash
pip install flask
pip install requests
```

Run program
```bash
python3 flasktest.py
```

Deactivate venv
`deactivate`
